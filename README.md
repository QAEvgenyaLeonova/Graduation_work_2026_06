# Автотесты для сайта «Читай-город»

Проект содержит набор автоматизированных тестов для проверки функциональности сайта интернет-магазина «Читай-город» (chitai-gorod.ru). Тесты покрывают сценарии поиска книг, валидацию элементов интерфейса и базовые проверки API.

## Задача проекта

Основная цель — обеспечить стабильность критически важных пользовательских сценариев:
*   Проверка корректной работы поиска по названию и автору (в том числе с учетом кириллицы и латиницы).
*   Валидация отображения карточек товаров и текстовых полей.
*   Проверка обработки случая «ничего не найдено».
*   Базовые проверки доступности эндпоинтов API и структуры ответов.
*   Автоматизация рутинных проверок UI для ускорения регрессионного тестирования.

## Структура проекта

| Файл / Папка | Назначение |
| --- | --- |
| `chitai_gorod_tests/pages/` | Реализация паттерна Page Object. Содержит классы, инкапсулирующие логику взаимодействия со страницами (например, `catalog_page.py`). |
| `chitai_gorod_tests/tests/` | Тестовые сценарии на базе pytest. Разделены на UI-тесты (`test_ui.py`) и API-тесты (`test_api.py`). |
| `conftest.py` | Фикстуры pytest. Содержит логику инициализации браузера (Chrome), настройки аргументов и управление жизненным циклом драйвера. |
| `config.json` | Конфигурационные параметры: базовый URL сайта, базовый URL API и таймауты. |
| `test_data.json` | Тестовые данные: список фраз для поиска и фраза для проверки отсутствия результатов. |
| `requirements.txt` | Список зависимостей проекта (selenium, pytest, requests, allure-pytest). |
| `pytest.ini` | Настройки запуска pytest: пути к тестам, маркеры (`ui`, `api`) и параметры генерации отчета Allure. |

## Как запустить тесты

### 1. Подготовка окружения

1.  Убедись, что на компьютере установлен Python версии 3.8 или выше.
2.  Установи необходимые зависимости:
    ```bash
    pip install -r requirements.txt
    ```
3.  Убедись, что у тебя установлен Google Chrome. Драйвер (chromedriver) подтягивается автоматически через Selenium (версии 4.6+), если он совместим с твоей версией браузера.

### 2. Запуск всех тестов
